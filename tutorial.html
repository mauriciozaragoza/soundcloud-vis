Setup

First of all you have to make sure you have installed the needed programs in your computer.
These are the programs that are recommended to have installed before continuing:

  • Python
  • Python Pip
  • MongoDB
  • PyMongo
  • Soundcloud API
  • Bootstrap (optional)

Since Bootstrap can be pulled from the internet there is no real need to install it, however you DO need to install all the other software in order to successfully use Soundcloud + Mongo
NOTE: This tutorial uses Python for database conection, data analysis and Soundcloud API but it can be implemented on any other language you may want.
The Soundcloud API even provides code for Ruby, Javascript and/or PHP

This section is mainly about the Soundcloud API used by developers to link and use Soundcloud on any site. The API page already contains a lot of examples you may need to develop a fine application on your own but still you may want to tailor it to better suit your needs. The examples posted on the API Reference page may give a better insight on what you really want to do.

MongoDB Connection

Now to configure the database connection you need to specify the location of the folder that will contain the database:
mongod --dbpath /data/db
and after that you need to keep an open terminal that will work as the server while on another terminal the connection is made running:
mongod

Soundcloud Data Manipulation

After all these steps we are finally ready to do data manipulation on Soundcloud information. Using a GET instruction the simple server calls the Soundcloud database and fetchs information according to the genre filter used, if nothing is inserted then the lastest uploaded songs will be fetched.
As you can tell you need to first fire up the simple server in the terminal and then make the query. Here the database is used as means of cache memory and nothing else.
